# 🚦 RoadSafetyHub

RoadSafetyHub is a Java-based web application designed to promote road safety awareness and manage safety-related data efficiently. The application allows users to interact with road safety services through a simple and user-friendly interface.

## 📌 Features

- User registration and login  
- Road safety awareness information  
- Data storage using MySQL  
- Dynamic web pages using JSP and Servlets  
- Form validation and exception handling  
- Database connectivity using JDBC  

## 🛠️ Tech Stack

- Core Java  
- Servlets  
- JSP  
- JDBC  
- MySQL  
- HTML  
- CSS  
- Apache Tomcat  
- Eclipse IDE
